# HTML Guanabara

Bem-vindo ao meu repositório de estudos de HTML e CSS!

Este repositório serve para documentar e compartilhar minha evolução no aprendizado de HTML e CSS através do curso em vídeo do Guanabara. É um espaço simples, mas espero que seja útil para outros aprendizes e interessados no tema.

## Sobre

Aqui você encontrará:

- **Desafios:** Exercícios e desafios realizados durante o curso.
- **Projetos:** Pequenos projetos desenvolvidos para praticar e aplicar os conhecimentos adquiridos.
- **Anotações:** Anotações e referências que considero importantes para o aprendizado.

## Observações

- O módulo 1 contém menos conteúdo, pois muitas das aulas abordavam tópicos que eu já conhecia. Portanto, foquei mais nos desafios.
- Fique à vontade para explorar e utilizar qualquer material deste repositório.
- Se encontrar algo faltando ou tiver sugestões, sinta-se livre para contribuir!

## Agradecimentos

Espero que gostem do meu repositório e que ele seja útil em sua jornada de aprendizado. :D
